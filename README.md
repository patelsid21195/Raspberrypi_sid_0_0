# 🤖 My First Robot Project

## 📌 Overview

This project documents my journey of building a mobile robot using Raspberry Pi.

The goal is to learn:

* Embedded programming
* Motor control
* Sensors & feedback (encoders)
* Control engineering basics

---

## 🎯 Objectives

* Build a 2-wheel robot platform
* Control motors using Raspberry Pi
* Read encoder data for feedback
* Implement basic motion control

---

## 🧰 Hardware Components

### Core

* Raspberry Pi 5
* MicroSD card (64GB)
* Power supply (official)

### Motion

* DC geared motors with encoder (251 RPM, 12V)
* Motor driver (TB6612FNG)
* Wheels (65mm)

### Electronics

* Breadboard
* Jumper wires (Dupont cables)
* (To buy) Battery pack (7.4V LiPo or 6x AA)
* (To buy) Multimeter
